Étape 01

Blocages

Ce que personne ne te dit : les blocages trop extrêmes ne marchent pas. Quand TikTok est limité à 45 min, tu vas les renta au max. C'est l'effet de rebond — exactement comme une dopamine detox.

La bonne approche : blocages programmés

Pas de limites de temps. Des plages horaires bloquées. Tu veux pas te retrouver sans accès à Insta quand t'en as vraiment besoin.

  • →Moments stratégiques :  nuit, sortie des cours, matin, sport, quality time
  • →Incontournables :  la nuit — TOUT sauf les apps vitales
  • →À bloquer :  Instagram, TikTok, YouTube (distractions)
  • →Pas trop extrême :  garder Insta visible sur la homepage — sinon la FOMO crée encore plus d'envie

Widgets Scriptable

Installe Scriptable et demande à Claude de te coder n'importe quel widget. Utilise les stacks pour en avoir plein tout en gardant ta homepage minimaliste.

⏱ Heures restantes dans l'année
▼
Affiche en temps réel combien d'heures il reste dans l'année. Pour rester motivé.
let now = new Date();
let endOfYear = new Date(now.getFullYear(), 11, 31, 23, 59, 59);
let diff = endOfYear - now;
let hoursLeft = Math.floor(diff / (1000 * 60 * 60));

let widget = new ListWidget();
widget.backgroundColor = new Color("#0F0F0F");
widget.addSpacer();

let text = widget.addText(`${hoursLeft}h`);
text.textColor = new Color("#FFFFFF");
text.font = Font.boldSystemFont(36);
text.centerAlignText();
widget.addSpacer();

Script.setWidget(widget);
Script.complete();

💪 Programme semaine + séance du jour
▼
Ces deux widgets fonctionnent ensemble — place-les côte à côte sur ta homepage. L'un affiche ton programme de la semaine (lundi : tirage, mardi : run…), l'autre zoome sur la séance du jour avec le détail des exercices. Donne ton programme à Claude pour qu'il adapte les deux codes.
Planning semaine Tous les jours d'un coup d'œil — jour + séance associée.
Séance du jour Détail complet des exercices du jour automatiquement.

Code — Planning semaine

const program = [
  { day: "lundi",    work: "dips lourd + renfo pull" },
  { day: "mardi",    work: "run 10-15km + legs" },
  { day: "mercredi", work: "repos" },
  { day: "jeudi",    work: "dips volume + renfo pull" },
  { day: "vendredi", work: "run 9min + pk ffa" },
  { day: "samedi",   work: "pk lsc + upper" },
  { day: "dimanche", work: "long run + mobi/renfo" }
];

const widget = new ListWidget();
widget.backgroundColor = new Color("#ffffff");
widget.setPadding(18, 18, 18, 18);

function customFont(name, size, fallback) {
  try { return Font.custom(name, size); }
  catch { return fallback; }
}

const dayFont = customFont("Montserrat-SemiBold", 16, Font.boldSystemFont(16));
const workFont = customFont("Montserrat-Regular", 16, Font.systemFont(16));
const black = new Color("#000");

for (let item of program) {
  const row = widget.addStack();
  row.layoutHorizontally();
  row.centerAlignContent();
  const day = row.addText(item.day + " : ");
  day.font = dayFont;
  day.textColor = black;
  const work = row.addText(item.work);
  work.font = workFont;
  work.textColor = black;
  widget.addSpacer(5);
}

if (config.runsInWidget) {
  Script.setWidget(widget);
  Script.complete();
} else {
  await widget.presentMedium();
}

Code — Séance du jour

const widget = new ListWidget()
widget.backgroundColor = new Color("#ffffff")
widget.setPadding(18, 18, 18, 18)

function customFont(name, size, fallback) {
  try { return Font.custom(name, size) } catch { return fallback }
}

const titleFont = customFont("Montserrat-Bold", 18, Font.boldSystemFont(18))
const subFont = customFont("Montserrat-SemiBold", 16, Font.semiboldSystemFont(16))

const black = new Color("#000")

const daysFR = ["Dimanche","Lundi","Mardi","Mercredi","Jeudi","Vendredi","Samedi"]

// MODIFIE AVEC TON PROGRAMME
const sessions = {

  "Mardi":    { name: "Run 10-15km + legs",       detailsKey: "Legs" },

  "Jeudi":    { name: "Dips volume + renfo pull", detailsKey: "DipsVolume" },
  "Vendredi": { name: "Run 9min + Parkour FFA",   detailsKey: null },

📊 % de la journée restante
▼
Affiche le pourcentage de la journée qu'il reste. Brutal et efficace.
let now = new Date();
let elapsedSeconds = now.getHours() * 3600 + now.getMinutes() * 60 + now.getSeconds();
let remainingSeconds = 86400 - elapsedSeconds;
let percentRemaining = (remainingSeconds / 86400) * 100;

let widget = new ListWidget();
widget.backgroundColor = new Color("#0F0F0F");
widget.addSpacer();

let main = widget.addText(`${percentRemaining.toFixed(1).replace(".", ",")}%`);
main.textColor = new Color("#FFFFFF");
main.font = Font.boldSystemFont(36);
main.centerAlignText();

let sub = widget.addText("de la journée restante");
sub.textColor = new Color("#AAAAAA");
sub.font = Font.systemFont(14);
sub.centerAlignText();

widget.addSpacer();
Script.setWidget(widget);
Script.complete();

Étape 05

Homepage & Système

  • →Minimaliste :  tout sur UNE page. Monochrome, noir et blanc. Pas de surcharge visuelle.
  • →Apps utiles devant :  réfléchis à ce que tu utilises le plus. Le reste dégage.
  • →Wallpaper :  soit une personne qui te motive, soit une photo qui te déter, soit rien.
  • →Personnalise :  Installe l'app Color Widget pour personnaliser tes widgets. Un texte sur ton lockscreen. En un regard tu es déterminé.
  • →Notifs :  coupe tout ce qui est inutile. Chaque notif inutile = une stimulation pour rien.
Important : ne supprime pas Insta de ta homepage. C'est trop extrême — la FOMO va te faire aller encore plus dessus pour "rattraper" ce que tu louperais.

Étape 08

Moments Off Programmés

Transformer son téléphone, c'est aussi savoir le poser.

  • →Réserve des plages dans ta semaine où le téléphone est complètement posé
  • →Balade sans téléphone. Sport sans musique. Moments de qualité avec les gens.
  • →Le meilleur moment : le soir — pose ton téléphone hors de ta chambre ou loin du lit
  • →20-30 minutes sans écran avant de dormir et après le réveil
Pourquoi c'est critique : si tu niqu*s tes niveaux de dopamine dès le réveil ou juste avant le sommeil, ta journée et ton sommeil seront forcément de mauvaise qualité. C'est pas négociable.
Étape 09

Créer

Beaucoup de gens pensent qu'ils sont pas créatifs. C'est faux — ils passent juste plus de temps à consommer qu'à créer.

Mon expérience

J'étais à court d'idées sur Instagram. Je scrollais des heures sans trouver l'inspiration. Puis j'ai réalisé que ça venait du fait que je créais plus assez. Je me suis forcé à faire des vidéos. Les premières étaient vraiment nulles — pas un peu, vraiment. J'avais envie d'arrêter. Mais j'ai continué. Et là quelque chose a changé : le contenu que je consommais s'est mis à m'inspirer.

La vraie transformation arrive quand tu crées, ton cerveau est en mode "recherche". Une vidéo que t'aurais quittée en 10 secondes, tu la regardes jusqu'au bout. Moi ça m'est arrivé des dizaines de fois : un truc que j'ai regardé m'a inspiré 6 mois plus tard. C'est ça qui fait que même le temps que tu passes sur ton téléphone travaille pour toi.

Apps pour créer depuis ton téléphone

  • →CapCut / Edits — montage vidéo
  • →Canva — visuels, stories, posts
  • →BandLab — musique
  • →Ou juste un papier et un crayon. L'important c'est de créer.
